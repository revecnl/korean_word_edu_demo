# Interactive Korean Learning Demos

초급 한국어 학습자가 화면을 직접 조작하며 문장을 만들어 볼 수 있도록 제작한 웹 기반 활동 프로토타입입니다.

## Included demos

- **장바구니 게임**: 물건, 수량, 단위 명사, 가격 표현 연습
- **날짜와 요일 연습**: 날짜, 요일, 일정 질문과 대답 연습
